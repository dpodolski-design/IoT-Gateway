# Документация проекта IoT Gateway

Прототип модуля IoT-интеграции для платформы TAS (Telephony Application Server): связь умного дома с телеком — уведомление о звонках на колонки и инициация звонков по событиям (например, датчик дыма).

## Быстрый старт

1. Установить зависимости: `pip install -r requirements.txt`
2. Настроить `.env` (см. [03 — Установка и настройка](03-installation.md))
3. Создать БД и выполнить `scripts/init_db.sql`
4. Запустить: `uvicorn iot_gateway.main:app --reload --port 8000`
5. Открыть http://localhost:8000/docs для интерактивной документации API
